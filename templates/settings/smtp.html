Quick Reference

Common SMTP Ports
  • Port 25: Standard SMTP (often blocked by ISPs)
  • Port 587: STARTTLS submission (recommended)
  • Port 465: SSL/TLS direct connection
  • Port 2525: Alternative submission port
Popular Providers
  • Gmail: smtp.gmail.com:587 (STARTTLS)
  • Outlook: smtp.live.com:587 (STARTTLS)
  • Yahoo: smtp.mail.yahoo.com:587 (STARTTLS)
  • SendGrid: smtp.sendgrid.net:587 (STARTTLS)
Security Notes
  • Always use encrypted connections in production
  • Enable 2FA and use app passwords when available
  • Monitor for failed authentication attempts
  • Regularly rotate SMTP credentials
